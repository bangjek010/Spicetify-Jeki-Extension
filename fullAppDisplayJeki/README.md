# Full App Display Jeki

Fancy artwork, track status, custom waveform and lyrics display overlay.

## Features

- **Immersive Full-Screen Display**: Beautiful cover art with ambient background blur.
- **Interactive Waveform Progress Bar**: Renders audio analysis data as a waveform, allowing you to seek by clicking or dragging.
- **Track Peek**: Shows previews of previous and next tracks in the queue when hovering.
- **Lyrics Integration**: Integrated lyrics display with active line highlighting and scrolling.
- **Customizable Layouts**: Renders controls and details in a clean, modern interface.

## Installation

Copy the `fullAppDisplayJeki.js` file to your Spicetify extensions directory:
`%appdata%\spicetify\Extensions`

After copying, register and apply the extension:

```bash
spicetify config extensions fullAppDisplayJeki.js
spicetify apply
```
