# Spicetify Jeki Extensions

A collection of custom Spicetify extensions to enhance your Spotify desktop experience.

## Extensions Included

### 1. Mini Lyric Customized
A floating Picture-in-Picture lyrics window that stays on top of other applications.
- **File**: `Mini-Lyric-Customized/Mini-Lyric-Customized.js`
- **Features**: Custom themes (Spotify, Pink, Kawaii, Ocean, Sunset, etc.), adjustable font size, player controls, and lyrics centering.
- Read more: [Mini Lyric Customized README](Mini-Lyric-Customized/README.md)

![Mini Lyric Preview](Mini-Lyric-Customized/Popup%20Mini%20Lyric%20Wide.png)

### 2. Adblock
Blocks all audio and UI advertisements on Spotify.
- **File**: `adblock/adblock.js`
- **Features**: Active ad blocking, auto-mute fallback for audio ads, statistics tracker, and a custom popup configuration modal.
- Read more: [Adblock README](adblock/README.md)

### 3. Full App Display Jeki
A beautiful, immersive full-screen display for your active track and lyrics.
- **File**: `fullAppDisplayJeki/fullAppDisplayJeki.js`
- **Features**: Large artwork display with ambient background blur, interactive wave progress bar, track peek, and integrated lyrics display.
- Read more: [Full App Display Jeki README](fullAppDisplayJeki/README.md)

![Full App Display Preview](fullAppDisplayJeki/FullAppDisplay%20With%20Lyric.png)

## Installation

To load these extensions in Spicetify, copy the respective `.js` files to your Spicetify extensions directory:
`%appdata%\spicetify\Extensions`

After copying, run:

```bash
spicetify config extensions <extension-name>.js
spicetify apply
```
