# Mini Lyric Customized

Creates a floating, Picture-in-Picture style lyrics overlay window that stays on top of other apps.

## Features

- **Floating Overlay**: Keep track of lyrics while working in other applications.
- **Multiple Themes**: Supports Spotify Green, Pink Pop, Kawaii, Ocean Blue, Sunset, Galaxy, luxury Gold, Cyberpunk, and more.
- **Interactive Controls**: Play, pause, skip, like, adjust volume, and change font size directly from the overlay.
- **Lyric Customization**: Adjust font size, toggle visibility, and align text (center/left).

## Installation

Copy the `Mini-Lyric-Customized.js` file to your Spicetify extensions directory:
`%appdata%\spicetify\Extensions`

After copying, register and apply the extension:

```bash
spicetify config extensions Mini-Lyric-Customized.js
spicetify apply
```
